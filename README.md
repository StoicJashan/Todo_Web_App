#  Todo Web App

A simple and interactive Todo Web App built with **Python** and **Streamlit**. It allows users to manage daily tasks through a clean and user-friendly interface.

##  Live Demo

🔗 https://stoicjashan-todo-web-app-web-qbi9g7.streamlit.app/

---

##  Features

*  Add new tasks
*  Mark tasks as completed
*  Live task counter
*  Balloons animation when all tasks are completed
*  Automatically saves tasks
*  Clean and minimal user interface

---

## 🛠️ Built With

* Python
* Streamlit
